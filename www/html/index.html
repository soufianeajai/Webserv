About Webserv

Webserv is a lightweight, custom-built HTTP server project developed in C++. It is designed to teach the fundamentals of web servers, networking, CGI, and more.

Key Features

  • Supports HTTP/1.1 protocol
  • Handles GET and POST requests efficiently
  • Custom CGI support for dynamic content
  • Customizable error handling and pages
  • Easy to extend and configure
  • Fast and lightweight
